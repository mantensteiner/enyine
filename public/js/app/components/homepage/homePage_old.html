What is enyine?

enyine is designed to organize people who know that action is the foundation to achieve desired results. All the best thoughts don't mean much, if nobody is empowered to make them reality. enyine offers following approaches for helping you to achieve your goals.

  • Flexible Planning and Tracking. Plan your next steps by using a system of customizable work items which consist of Item Types, Items, Status and Tags. Enter your actionable data with customizable Value records, for example time records. We keep a full Change Log on all actions in case you need to look something up. Use the drag&drop enabled Board of Items by Status to quickly grasp & act on the current project situation. Use rich querying capabilities, Filters and inline Aggregations with Chart-Visualizations to get insights out of your data.

    The power lies in simplicity & flexibility, as you find your optimal project structure along the way and dynamically grow it with your needs. You may want to start with a project Template, but evolve and adapt the work items to fit exacly for your process. With building blocks tailored to your project you are able to improve your workflow for more effective results.
  • Smooth Information Management for Teams. Sometimes information doesn't fit in a task or another work item - you just want to be creative and write free text (in Markdown) on a project matter. Therefore Notes can be created for just yourself (private), maybe as an idea scratchpad, or public for the whole team, maybe to document a strategic document like a roadmap.

    Also Comments can be added to let information flow between team members and discuss directly on Items. These Comments are visible in context of Items or Notes, but also in a global Timeline which is compareable to a team chat application.

    For getting external references or resources into the project, Urls can be conveniently stored for later referral and edited inline. We also provide a automated Browser-link Importer and short-cut data entry with 'quick add' options. Each page can be stored as Internal Bookmark for quick access later on.
  • Connecting Tools and Systems. Use the right tools for the job by using the power of Integrations. Github as repository for your content & work items. Slack for team chat, document & file mangagement and much more. Items & Values can be automatically created from Github commits, changes & comments can be forwarded to your projects Slack-channel.

    With configurable Webhooks enyine is readily equipped to automate the data flow to and from any external application. This powerful concept allows you to collect all kinds of data, from all kinds of sources and devices.

    enyine can be the 'data management glue' that holds your services & applications together - but only if you like to do so!

Building Blocks

Spaces

Spaces are the core building block of enyine. A user is a member of one or more spaces and has access to the data inside. A space could be a project and you can create spaces for yourself or invite as many members as you like.

Items

Items bring structure to your projects. A item can be a feature, bug, meeting, idea or whatever you want.

Values & Value Types

Tracking of actionable data - data which allows for further analysis - without much hassle is a core goal of enyine. Focus on your actions and spend minimum effort on manual data input. Going further, you can use different value types & units to measure anything.
